> Do something to see the posts of a discussion.

## Install

```sh
composer require nearata/flarum-ext-dsts:"*"
```

## Update

```sh
composer update nearata/flarum-ext-dsts:"*"
php flarum cache:clear
```

## Remove

```sh
composer remove nearata/flarum-ext-dsts
```
